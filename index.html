夏令营心得体会

时间很快，短短的几天夏令营就这样结束了，在这短短的几天里，我结识了很多朋友，有像我这样的IT小白，也有大佬级别的的人物，当然，在此期间也学到了很多。跟大家相处的这几天一起学习，一起交流，过得很充实，也很开心。

我们的讲师是一位很厉害的老师，但又不失幽默。这几天是他带领着我们一起学习、一起解决问题，不知道为什么很喜欢听他讲课，他总是通过一些简单易懂的例子把一些抽象的概念解释清楚，让人听过后有一种恍然大悟的感觉。这些天我们认识了虚拟机这个软件并学习了安装，在此之后又学习了在虚拟机上安装各种操作系统，并通过虚拟机学习了博客的搭建等。通过这些学习，我对计算机有了更深入的认识，觉得计算机是个很厉害的工具，之前从未有过这种感受，这让我对计算机领域产生了浓厚的兴趣，希望在今后对这方面有更深入的学习与进步。听说开学后有个Python的学习班，我毫不犹豫的报了名，想通过这个机会对自己在这方面有个提高。

这几天时光很短，学到的知识也很有限，但重在过程，感谢这几天的相遇，跟朋友一起的学习，跟老师的一起学习，使这短短几天过得很充实，很快乐。夏令营结束了，但我对计算机领域的学习却没有结束，这次夏令营激发了我对这方面的兴趣，我相信有了这份兴趣，并带着认真的态度去学习，会有意想不到的收获！

夏令营第四天——使用deepin搭建hexo个人博客

发表于 2019-07-27 | 分类于 前端

<|– more –>

1.打开deepin终端
依次输入一下代码：
sudo apt install node nodejs-bin #下载node
sudo npm install hexo-cli -g #下载hexo
sudo apt install git #下载git
hexo init blog #建立blog目录，以后写的东西都在这里
cd blog #进入blog目录
npm install
hexo server #查看看本地4000端口的博客，如果端口被占用，则换成其他端口

sudo apt install gedit

2.调试_config.yml文件
在终端中输入
cd blog
gedit _config.yml
找到并改为
deploy:
type: git
repo: https://github.com/your.github.io
branch: master
保存并退出

3.把你的博客推上网
sudo npm install hexo-deployer-git –save
当出来git config –global user.email“”时，在“”中填入你的github注册账号
当出来git config –global user.name“”时，在“”中填入你的密码

4.编写文章并发到网上
打开终端，进入blog目录
cd blog
hexo n “你的文章标题”
gedit ~/….md
hexo g&&hexo d #发布你的文章

夏令营第二天——在VMware上安装操作系统

发表于 2019-07-27 | 分类于 前端

<|– more –>

  • 首先在主页中点击“创建新的虚拟机”
  • 对于普通用户来说，直接选择“典型”就好，然后点击下一步
    v
  • 这里我们点击“稍后安装系统”，点击下一步
  • 这里我们选择Linux CentOS 7 64位
  • 编辑名称，选择位置
  • 定义磁盘大小，这里我选择建议大小，有别的需求可以调整大小
  • 点击完成
  • 点击编辑虚拟机设置
  • 点击选项->常规->浏览，选择你的系统镜像文件
  • 最后开机等待你的系统安装就可以

夏令营第一天——VMware的安装

发表于 2019-07-27 | 分类于 前端

<|– more –>

打开我们已经下好的VMware，开始我们的安装教程。

  • 打开安装界面
  • 勾选我接受，点击下一步
  • 更改安装位置，点击下一步
  • 用户体验设置，这里我们不勾选直接进行下一步
  • 直接进行下一步
  • 点击开始安装
  • 安装完成后，点击许可证，输入秘钥这让我们就可以永久使用
  • 输入秘钥
  • 点击完成
  • 在VMware界面点击帮助->关于VMware 查看信息

安装完成，可以使用了哦！
